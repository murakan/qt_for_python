# Installation
1. Prepare pipenv.
1. Use pipenv to install related packages at each directories.
1. Launch main.py.

## Widget pattern
````
$ cd widgets
$ pipenv install
$ pipenv run python main.py
````

## QML pattern
````
$ cd qml
$ pipenv install
$ pipenv run python main.py
````
